# Proyecto TNK

Juego 2D singleplayer de acción con estructura de expediciones.

El jugador parte desde una aldea para realizar runs de combate en escenarios cerrados.
Durante cada run se obtienen recursos y mejoras temporales.
Al regresar a la aldea, esos recursos se utilizan para desbloquear mejoras permanentes.

El progreso entre runs es persistente y forma parte del núcleo del juego.
Las runs son cortas y se centran en movimiento, posicionamiento y uso de armas.

El objetivo del proyecto es construir un MVP con un loop jugable claro y sólido
antes de añadir complejidad o contenido adicional.
